General Information

What is OpenG Builder?
The OpenG Builder is a general purpose build tool for creating software products written in G within the LabVIEW Development Environment. Read our PRODUCTNAME overview.
Who should use OpenG Builder?
Anyone who has software engineering needs that require a build tool that is open, extensible, and flexible. OpenG Developers use the OpenG Builder for building all of the OpenG software products .

Getting Started

What is OpenG Commander?
The OpenG Commander is a LabVIEW IDE Extension which provides a Graphical User Interface utility for downloading and installing online OpenG Packages. For additional technical terms, see the glossary.
What is an OpenG Package?
An OpenG Package is a file that contains everything you need to install a LabVIEW reuse library or IDE extension. For additional technical terms, see the glossary.

Building Applications

How do I do Build a Stand-Alone Executable?
The OpenG Builder allows one to build an executable by optionally converting one of the destination LLBs into an executable (EXE) file. If this destination folder contains a Top-Level VI, then the EXE will auto-launch the Top-Level VI.

Troubleshooting

After installing the OpenG Builder I don't see an "OpenG Builder" item in the LabVIEW Tools menu. What's wrong?
You probably need to restart LabVIEW in order for the menus to be refreshed. Restart LabVIEW and check again.
