Installation

What other software must be installed first?
Before you can install this product, you must install the following packages:
  • LabVIEW Development Environment (6.1 or Greater for Windows, Mac OS X, or Linux)
  • LabVIEW Application Builder (only needed if building stand-alone executables)
  • OpenG Commander
How do I install OpenG Builder?
Please follow these steps:
  1. Run the OpenG Commander
  2. Download and Install the OpenG Builder package (ogrsc_builder)
  3. Restart LabVIEW
  4. Launch the OpenG Builder from the LabVIEW Menu at Tools >> OpenG Builder
How can I uninstall PRODUCT-NAME?
  1. Run the OpenG Commander
  2. Uninstall the OpenG Builder package (ogrsc_builder)
  3. Restart LabVIEW
