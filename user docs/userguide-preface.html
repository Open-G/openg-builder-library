Description of the Document

This guide describes the Graphical User Interface (GUI) and Application Programming Interface (API) of the OpenG Builder. Error log messages and file formats are listed in appendixes. Additional sources of documentation are listed at the end of this Preface.

Audience

This guide is intended for LabVIEW developers who have a working knowledge of the LabVIEW Application Builder. The OpenG Builder was designed by and for software engineers with a need to implement a fully automated build process. The OpenG Builder provides a multitude of options which may not be useful for the casual user whose only goal is to distribute thier application as a stand-alone executable. For these users we recommend the LabVIEW Application Builder.

Multi-Platform Release

The OpenG Builder is compatible on all platforms which run LabVIEW 6.1 or greater. It has been tested on LabVIEW 6.1, 7.0, and 7.1 for Windows, Mac OS X, and Linux.

LabVIEW Documentation

Documentation for LabVIEW and the LabVIEW Application Builder is distributed by National Instruments. This document assumes that you have access to these.
