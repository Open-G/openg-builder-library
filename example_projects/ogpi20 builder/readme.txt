Installation:
Copy .deab files and "ogpi_buider.vi" file to <ogpi20> root directory (the directory already
contains build_ogpi.vi)

Run:
Run the "ogpi_buider.vi". A target built will be saved in <ogpi20\built\installer> directory.


NOTES
1) Make sure that you have the last DEAB release installed
2) It was tested in W2K only
